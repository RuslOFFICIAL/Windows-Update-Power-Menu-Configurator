Windows-Update-Power-Options (WUPO) is a simple project to bypass forced Windows update restart and shut down.
This project also includes basic method for compiling a file to .EXE file.
I hope someone will find it useful for th-N2em3elves. :3

Version (N1.N2-N3.N4) meaning:
- N1 means a noticeable-big change in Program folder.
- N2 means a small change which can be noticed in the code in Program folder.
- N3 means a noticeable-big change in Compile folder.
- N4 means a small change which can be noticed in the code in Compile folder.
